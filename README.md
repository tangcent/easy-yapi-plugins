# easy-yapi-plugins
easy-yapi插件仓库

## 安装

- [Download](https://github.com/tangcent/easy-yapi-plugins/raw/master/idea/easy-yapi.jar) the plugin jar and select "Install Plugin From Disk" in IntelliJ's plugin preferences.
